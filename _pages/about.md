## 关于我

我目前是 华盛顿大学圣路易斯分校（Washington University in St. Louis） 金融科技与分析（Financial Technology and analytics）硕士研究生，本科毕业于 宾夕法尼亚州立大学（The Pennsylvania State University），主修计算机工程（Computer Engineering）。

我的主要兴趣方向是 数据分析/开发，同时具备扎实的软件与系统工程背景。我关注如何将工程能力与数据分析方法相结合，从真实业务或系统数据中提取信息，并通过分析与建模支持问题定位和决策。

在本科与研究生阶段的学习和项目实践中，我在 数据分析、数据处理与系统开发 等方向积累了较为全面的经验，包括对日志数据和传感器数据的整理与分析以及开发支持数据展示与分析的后端服务和可视化仪表盘。此外，我也具备一定的机器学习与深度学习实践经验，曾使用 PyTorch 实现并评估图像超分辨率相关模型。

在实习经历方面，我曾在 上海椰岛游戏（Coconut Island Games） 担任游戏测试实习生，主要负责基于性能与运行数据分析游戏中的性能瓶颈与异常问题，并参与回归测试和性能测试流程；同时也曾在 江苏狄诺尼信息技术有限公司 担任软件研发实习生，参与使用 C++ 与 Python 实现 CAD 自动化绘图功能，并结合大语言模型（LLM）API 完成相关系统集成工作。

目前，我正在积极寻找 数据分析、数据科学或与数据相关的软件/系统方向实习或项目机会，希望将自身的工程背景与数据分析能力相结合，参与解决真实场景中的问题，并持续提升在数据与智能技术方向的实践能力。
